# Vedic Dashboard

An interactive Streamlit application for Vedic Astrology calculations.

## Features
* **Dagdha Rashi:** Calculates burnt signs based on birth Tithi.
* **Badhak:** Identifies obstruction houses/planets based on Ascendant.

## How to Run Locally
1. Install dependencies:
   `pip install -r requirements.txt`
2. Run the app:
   `streamlit run vedic_dashboard_v9.py`
